Add New Feedback Form:
1. Copy "PROJECT Feedback (Example Template)" form
2. Update the title and description (including links), category options, and anything else specific to the project
3. Link responses to UFBS Google Sheet
4. Add response Sheet URL to description
5. Update Settings > Presentation > Confirmation message
6. Format response Sheet tab like template
7. Add Form URL to response Sheet


Afterwards:
- Add Form URL on project webpage
- Lock and add Form URL on any old Google Docs
- Add Form and response Sheet URL to UFBS webpage
