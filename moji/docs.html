R74moji Docs


These are the docs for the HTML/Javascript R74moji library.

Add this to your <head> tag to load the script...
<script src="https://R74n.com/moji/load.js"></script>
You must load this script on any page that includes the emojis, or else they will not appear.

To add an emoji to your text, insert this code anywhere, replacing "emoji_type" with the code of the desired emoji found here...
<emoji type="emoji_type"></emoji>

When copied and pasted, the emojis will use the closest textual representation possible.

When hovered over, the name of the emoji will appear.
