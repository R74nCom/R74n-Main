Rue Changelog

This changelog only contains additions and major changes from v1.1 and above. Easter eggs, variants, and minor changes are not included. More can be found in the Explore with Rue Guidebook.

v1.3.1 - Aug 6, 2024 - Rue Chat

  • Rue Chat - Save your sessions in a standalone page!
  • New links: artfight, chat
  • Updated Rue examples

v1.3 - Jul 12, 2024

  • Commands can be typed like functions, like sort(b,c,g,a,e)
  • Unit conversion (e.g. 5 km to miles)
  • Argument "this" will be replaced by last highlighted text (e.g. translate this)
  • New commands:
      airport, bad eyesight mode, bing images, char, char count, clinical trial, color picker, count, dataset, decade, dice, email, fcc, flight, go blind, google web, http / https, imgops, ip, iplookup, karma decay, my name is, oid, patent, planecode, resolve, shuffle, sitesearch, spread, staring contest, stop blinking, streamer mode, tota11y, unfocus, urn, us census, weather, webaim, what's my, wikihow to, xkcd
  • Can now recognize:
      package tracking codes, flight numbers, aircraft codes, callsigns, IP addresses, R74moji shortcodes, HTML elements by #ID or .class, hashtags, Unicode codepoints, dice notation, OIDs, URNs, Multiplane codes
  • New small talk responses
  • Link redirects to newer R74n projects
  • Empty messages will close Rue if summoned by bookmarklet
  • Fixed: Doesn't respond to "who r u"
  • Fixed: Doesn't respond when math answer is zero
  • Fixed: QR code command
  • See more information in the Guidebook!
