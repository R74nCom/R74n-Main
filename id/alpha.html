Alpha Namespaces

For short strings of text, R74n uses the R74n Standard Alpha Namespaces to identify projects.

The Multiletter Alpha Codes are most often used, and can contain any letters A-Z, along with dashes (-). If needed, underscores (_) can replace them. For example, cpd represents the Copy Paste Dump.

Two-letter Alpha Codes are also documented, and contain exactly 2 letters A-Z. For example, sb represents Sandboxels.

They are used in the Universal Feedback System and X-R74n URN Namespace, among others.
