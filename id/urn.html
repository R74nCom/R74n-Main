URN Namespace

To link to certain projects, R74n uses Uniform Resource Names (URNs).

URNs follow this format:

urn:<namespace>:<string>:<...>

R74n URNs use the "X-R74n" namespace, and a standard Multiletter Alpha Namespace as the first string. Arguments can be added to the end to navigate to specific parts of a project, or to add additional functionality. For example:

urn:X-R74n:sandboxels:changelog

They can be redirected using the ID Resolver, where both "urn:" and "X-R74n:" can be left out. Arguments can be URL-encoded to include special characters, like spaces.
