Elemental on Discord

Elemental on Discord is an alchemy game inspired by Elemental 3 by carykh created in a Discord bot. Players start with 4 elements: Air, Earth, Fire, and Water. There are over 800,000 elements that can be created with just these 4, all voted on democratically by the community.

EoD was created by R74n on June 7, 2020, and later handed off to Nv7, who rewrote the bot with the ability to be added to any server. You can join the main instance with over 1600 members!

Players combine elements in channels like so: fire + water. The bot will add the result, Steam, to their inventory. There are over 1,000,000 combinations.
