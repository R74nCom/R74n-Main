Shapes

R74n Shapes is a set of generic shapes and icons, created at first for Infinite Chef.

They are drawn with a standard color palette, designed to be shifted in hue, saturation, and lightness.

#430000 - hsl(0°, 100%, 13%)
#700000 - hsl(0°, 100%, 22%)
#B61800 - hsl(8°, 100%, 36%)
#F03030 - hsl(0°, 86%, 56%) [Base]
#FF5E5E - hsl(0°, 100%, 68%)
#FF8383 - hsl(0°, 100%, 76%)

If you'd like to use Shapes in your project, please ask for permission!
